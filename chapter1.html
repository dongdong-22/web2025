Chapter1 Summary

자바스크립트의 특징
모든 웹 브라우저에서 작동
웹에서 실행결과 즉시 확인 가능
풀스택 웹 개발 뿐 아니라 다양한 용도의 프로그램 제작 가능
다양한 JavaScript 공개 API사용 가능
다양한 라이브러리와 프레임워크 사용 가능

!자바스크립트와 자바는 다른 언어임
넷스케이프 브라우저에서 사용 목적으로 개발됐으나, 이후 이름 변경(자바가 인기있었기에 그냥 이름 따온 것 뿐. 붕어빵에 붕어 없다.)
JavaScript vs Jscript -> ECMA 에서 JavaScript를 표준으로 지정. -> 현재 ES6(ECMASCript 2015)가 가장 많이 사용.

웹브라우저에서 F12키를 눌러 콘솔창을 띄우고 JavaScript 언어를 즉시 실행해볼 수 있다. (스크립트 언어)

인라인 스크립트? -> HTML 태그 안에 직접 작성. 간단한 명령을 처리할 경우 태그 안에 작성

내부 스크립트도 있음.

위 방법들은 수정이 용이하지 않음. 그래서 외부 스크립트를 연결해서 사용.

내부 스크립트? script코드 안에 작성.
