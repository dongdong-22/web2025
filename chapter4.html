Chapter4. 함수와 스코프

1함수
예시:
function greet() {
 console.log("Hello!");
}
과 같이 함수를 선언하면, 재사용성이 좋다. 익명 함수(Anonymous Function)
이름이 없는 함수 변수에 담거나 콜백 함수로 주로 사용
const greet = function() {
 console.log("Hi!");
};

즉시 실행 함수
정의하자마자 바로 실행되는 함수.
스코프(Scope)
변수가 유효한 범위.
전역 변수: 전체에서 접근 가능
var globalVar = 1;
지역 변수: 함수나 블록 안에서만 사용 가능
function test() {
 let localVar = 2;
}

var, let, const
var: 함수 스코프, 중복 선언 가능, 호이스팅됨
let: 블록 스코프, 중복 선언 불가
const: 블록 스코프, 선언과 동시에 초기화 필요, 값 변경 불가

화살표 함수(Arrow Function)

const sum = (a, b) => a + b;

콜백 함수(Callback)
다른 함수의 인자로 전달되어 실행되는 함수
function run(callback) {
 callback();
}
run(() => { console.log("콜백 실행"); });

호이스팅(Hoisting)
변수나 함수가 선언되기 전에 끌어올려지는 개념
console.log(x); // undefined
var x = 10;
console.log(y); // ReferenceError
let y = 20;
