Chapter2 summary

alert함수: 알림 창 표시 (확인용)

confirm함수: 확인과 취소를 누를 수 있음. 확인 -> True, 취소 -> False값 반환. 값 저장을 위해 변수 선언 필요

prompt함수: 값을 입력받을 때 사용. 첫 번째 인수는 내용, 두 번째 인수는 프롬프트 창의 입력값에 기본값에 저장된 값을 미리 작성해줌.

console.log 함수: 콘솔 창에 괄호 안의 내용들 표시함 -> 결과 표시 함수 중 주로 많이 사용

document.write 함수: 이것도 출력용


변수란?

변수는 값을 저장해주는 역할. let과 const를 통해 선언하는데, let은 할당 가능, const는 재할당 불가능

변수의 필요성: 변수를 사용하지 않고 변수값이 들어갈 부분마다 상수를 입력한다면 수정이 필요할 때 매번 모두 수정해줘야 됨. -> 변수 사용으로 간단화

자료형이란?

원시형, 복합형이 있다. 복합형은 array, object로 이루어져 있는데, 둘 다 객체라고 보면 된다.

typeof함수: 자료형 확인 가능 괄호는 감싸도 되고 안 감싸도 됨.
특수 기호 표시: 백슬래쉬를 활용하여 특수문자, 개행문자 등 사용 가능
출력 방법: 템플릿 리터럴을 이용해 간편하게 출력 가능 (파이썬의 f문자열과 비슷하다고 보면 됨.)
논리값: 0, "", NaN(Not a Number), Null, Undefined 5개는 False, 나머지는 True
Undefined는 선언 후 값 할당 않을 시 나오고, Null은 아예 없는 값이라는 뜻.(Null은 선언되지 않은 것을 출력하고자 할 때 나옴)
심볼: ES6에 새로 추가됨. 유일성을 보장. 객체에서 key로써 사용.
let id = Symbol()
const memeber = {
name = "Kim"
[id] = 1234
}
와 같이 사용, member[id] = 12345
