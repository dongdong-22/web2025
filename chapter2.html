Chapter2 summary

alert함수: 알림 창 표시 (확인용)

confirm함수: 확인과 취소를 누를 수 있음. 확인 -> True, 취소 -> False값 반환. 값 저장을 위해 변수 선언 필요

prompt함수: 값을 입력받을 때 사용. 첫 번째 인수는 내용, 두 번째 인수는 프롬프트 창의 입력값에 기본값에 저장된 값을 미리 작성해줌.

console.log 함수: 콘솔 창에 괄호 안의 내용들 표시함 -> 결과 표시 함수 중 주로 많이 사용

document.write 함수: 이것도 출력용


변수란?

변수는 값을 저장해주는 역할. let과 const를 통해 선언하는데, let은 할당 가능, const는 재할당 불가능

변수의 필요성: 변수를 사용하지 않고 변수값이 들어갈 부분마다 상수를 입력한다면 수정이 필요할 때 매번 모두 수정해줘야 됨. -> 변수 사용으로 간단화
