Chapter5. DOM(Document Object Model)

1. DOM이란?

DOM은 Document Object Model의 약자로, HTML 문서를 자바스크립트가 이해하고 조작할 수 있도록 "객체 트리 구조"로 표현한 것.

웹 브라우저는 HTML을 해석하면서 내부적으로 DOM 구조를 만들고, 자바스크립트는 이 DOM을 통해 웹페이지 요소에 접근하거나 변경할 수 있어요.

2. DOM 트리 구조 예시

예를 들어 다음과 같은 HTML이 있을 때:

  <body>
    <h1>제목</h1>
    <p>내용</p>
    <button>버튼</button>
  </body>

DOM은 이렇게 트리처럼 구성돼요:

  • document
    • html
      • head
      • body
        • h1
        • p
        • button

3. DOM을 이용한 JavaScript 조작

자바스크립트는 DOM을 통해 HTML 요소를 선택하고 조작할 수 있다.

  document.querySelector("h1").style.color = "blue";

→ 이는 <h1> 태그의 글자 색을 파란색으로 바꿔요.

4. 자주 사용하는 DOM 관련 메서드

  • document.getElementById(id) – ID로 요소 선택
  • document.querySelector(selector) – CSS 선택자로 하나 선택
  • document.querySelectorAll(selector) – CSS 선택자로 여러 개 선택 (NodeList 반환)
  • element.textContent – 요소 안의 텍스트 가져오거나 바꾸기
  • element.style – 요소의 스타일 속성 변경

5. DOM의 특징 요약

  • HTML 구조를 트리로 표현한 객체 모델
  • 브라우저가 자동으로 생성
  • JavaScript로 읽고 수정, 삭제, 추가 가능
  • 인터랙티브한 웹사이트의 핵심!
